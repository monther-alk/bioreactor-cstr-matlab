# Bioreactor Modeling & Simulation

A dynamic CSTR model simulating biomass growth and substrate 
consumption using Monod kinetics and mass balance equations, 
built in MATLAB.

## What it does
- Models steady-state and dynamic behavior of a continuous 
  stirred tank bioreactor
- Simulates the effect of dilution rate, feed concentration, 
  and kinetic parameters on reactor performance

## Key parameters
- μmax (max specific growth rate)
- Ks (half-saturation constant)
- Yield coefficient Y

## How to run
Open main_simulation.m in MATLAB and run. Adjust parameters 
in the constants section at the top.
